
# 제국 군주 카드 — "한눈에 이해" 리프레시 플랜

## 진단: 왜 지금 못 알아보는가

`src/components/empire/PackageUpgradeCards.tsx` 의 현재 표현은 신규/시니어 유저 진입을 막는 전문 용어 덩어리입니다.

| 현재 표현 | 유저 머릿속 |
|-----------|------------|
| "제국 군주 카드" | 게임인가? 등급? 무슨 카드? |
| "Recovery +60%" | Recovery? 복구? 환불? |
| "영토 +15%" | 땅을 사는 건가? |
| "Jackpot 풀 가중치" | 무슨 가중치… |
| "100 XP / 레벨" | XP가 뭐지 |
| "길드 전쟁 가중치 ×2" | 길드? 전쟁? |
| "패키지로 진화" | 어디서 진화? |

핵심 문제 3가지:
1. **무엇인가**가 한 줄로 없음 → 유저는 "이게 카드인지, 등급인지, 상품인지" 모름
2. **나에게 무슨 이득**인지 숫자 위에 직관적 한글 한 줄이 없음 (Recovery +60% → "손실 60%까지 보험금으로 자동 보상")
3. **어떻게 얻나**가 "Empire 패키지로 진화" 한 줄뿐 → CTA 부재

## 설계 원칙 — "1줄 정체 · 3줄 혜택 · 1버튼 행동"

```text
┌──────────────────────────────────────────┐
│ 👑 제국 군주 카드                          │
│ 입금하면 자동 지급되는 "혜택 등급 카드"     │ ← 정체 1줄
│ 등급이 오를수록 손실 보험·수익 가속·       │ ← 효과 1줄
│ VIP 룰렛 등 혜택이 강해집니다              │
│                                          │
│ [내 등급 보기 ↓]   [등급 비교 표 보기]    │ ← 명확한 CTA
└──────────────────────────────────────────┘
```

각 카드는 다음 5가지가 한눈에 보이도록 재구성:

```text
┌─────────────────────┐
│ 👑 제국 군주         │ ← 이름
│ [LEGENDARY · 5단계]  │ ← 등급 배지 + 단계 표시 (총 7단계 중 5)
│ ─────────────────── │
│ ✅ 손실 60% 자동 보상│ ← Recovery → 일상어
│ ✅ 보상 +15% 가속    │ ← 영토 → "보상 가속"
│ ✅ VIP 골드 룰렛 입장│ ← 잭팟 가중 → 실제 혜택
│ ✅ 출금 우선 처리    │ ← (실제 효과 그대로)
│ ─────────────────── │
│ 해금: 150만원 패키지  │
│ [💎 패키지 보기]     │ ← 잠금 카드에 직접 CTA
└─────────────────────┘
```

보유한 카드는:
```text
[LV.7 · 다음 단계까지 32 XP]
[████████░░░░ 68%]
[⚡ 지금 진화하기]   ← "진화" 옆 ⓘ 툴팁: "등급이 올라 혜택이 강화됩니다"
```

## 카피 매핑표 (전문 → 일상어)

| 전문 용어 | 새 표현 |
|-----------|---------|
| Recovery +X% | 손실 X% 자동 보상 |
| 영토 +X% | 보상 가속 +X% |
| Jackpot 풀 가중치 | VIP 골드 룰렛 입장권 |
| 길드 전쟁 가중치 ×2 | 길드 보상 2배 (선택 기능) |
| 골드 룰렛 + 가챠 | 매일 무료 골드 룰렛 1회 |
| 명예의 전당 +1티어 | 명예의 전당 자동 등록 |
| 100 XP / 레벨 | "100점 모으면 한 단계 진화" |
| 진화 | 단계 업 (강화) |

## 새 헤더 카피

```text
👑 제국 군주 카드 — 내 혜택 등급
입금하면 카드가 자동으로 강해지고, 강해질수록
손실 보험·수익 가속·VIP 혜택이 늘어납니다.

[?] 카드는 어디서 받나요?  ← 인라인 도움말 (toggle)
   └ Empire 패키지를 1회 입금하면 자동 지급됩니다.
     이후 미션·전투 보상으로 100점이 모이면 다음 단계로 진화합니다.

[?] 위험한가요?
   └ 카드는 상품이 아닙니다. 입금에 따라오는 "혜택 등급"이며
     별도 비용 없이 자동 강화됩니다.
```

## 잠금 카드 액션 강화

- `easy_50` 잠금 → **[50만원 패키지 보러가기 →]** → `/packages?focus=easy_50`
- `empire` 잠금 → **[Empire 패키지 보러가기 →]** → `/packages?focus=empire`

## 신규: "등급 비교 한 장" 다이얼로그

상단 [등급 비교 표 보기] 클릭 → Dialog 안에 7단계 × 4혜택(보험·가속·룰렛·우선출금) 매트릭스를 1장으로 표시. 모바일은 가로 스크롤 또는 세로 카드로 reflow. 시니어 모드(이미 `senior-mode` 클래스 존재) 대응.

## 시니어 모드 대응

- 현재 `.text-[10px]`, `.text-[11px]` 다용 → 시니어 모드에서 가독성 부족
- 본문 텍스트 최소 13px로 상향 (시니어 모드에선 자동 18px로 스케일)
- "Recovery", "XP", "LEGENDARY" 등 영문 단어는 시니어 모드일 때 한글 병기:
  - LEGENDARY → 전설(LEGENDARY)
  - XP → 진화점수(XP)
